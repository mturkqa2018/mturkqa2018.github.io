State Highway 28 (SH 28) is a State Highway in Kerala, India that starts in Kozhikode and ends at the state boundary. The highway is 103.6 km long.

The Route Map[edit]

Kozhikode - NH17 - SH 24 - Velluvambram - Manjeri junction - Nilambur - Chungathara - Edakkara - Vazhikadavu - State boundary
