SCM Anywhere
Developer(s)Dynamsoft Corporation
TypeRevision control
LicenseProprietary
Websitewww.scmsoftwareconfigurationmanagement.com

SCM Anywhere is SQL Server-based software configuration management tool with integrated revision control, bug tracking and build automation. It supports integration with CruiseControl.NET and ANT. Developed by Dynamsoft. 1

SCM Anywhere is a client/server system. The server manages a central database and a master repository of file versions. Users work on files in local client working folder, and submit changed files together in changesets. On-premises software and Software as a service editions available. 2

Integration with Visual SourceSafe compatible IDEs is supported as well as Cross-platform. 3
