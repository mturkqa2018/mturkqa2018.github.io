The Braille pattern dots-2356 ( â ¶ ) is a 6-dot braille cell with both middle and both bottom dots raised, or an 8-dot braille cell with both upper-middle and both lower-middle dots raised. It is represented by the Unicode code point U+2836, and in Braille ASCII with the number 7. 1

related 8-dot kantenji patterns

See also: Braille pattern dots-3678, Braille pattern dots-13678, Braille pattern dots-34678, and Braille pattern dots-134678

In the Japanese kantenji braille, the standard 8-dot Braille patterns 3678, 13678, 34678, and 134678 are the patterns related to Braille pattern dots-2356, since the two additional dots of kantenji patterns 02356, 23567, and 023567 are placed above the base 6-dot cell, instead of below, as in standard 8-dot braille. 5
