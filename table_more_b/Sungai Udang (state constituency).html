Sungai Udang is a state constituency in Malacca, Malaysia, that has been represented in the Melaka State Legislative Assembly. 1

The state constituency was first contested in 1974 and is mandated to return a single Assemblyman to the Melaka State Legislative Assembly under the first-past-the-post voting system. Since 2013, the State Assemblyman for Sungai Udang is Idris Haron from Barisan Nasional (BN). 2

Definition

The Sungai Udang constituency contains the polling districts of Bukit Terendak, Pekan Sungai Udang, Paya Rumput Jaya and Bertam Hulu. 3
